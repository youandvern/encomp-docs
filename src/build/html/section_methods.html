Section Methods

get_all_aluminum_section_sizes(section_type: str)

Returns a list of all available section sizes for the requested aluminum section type.

Parameters

section_type ("Angle" | "Channel" | "Circular" | "Rectangular" | "WF") – The name of the section type to retrieve

Returns

A list of all available section sizes for the requested type

Return type

list

>>> all_wf_sections = get_all_aluminum_section_sizes("WF")
>>> print(all_wf_sections)
["I 3 x 1.64", "I 10 x 10.3", "WF 5 x 6.49", "S 4 x 3.28" ...]

get_aluminum_section_properties(section_type: str, section_size: str) → dict

Returns a dictionary with physical properties of the requested aluminum section type and size. This method expects the section_size to match a Size in the respective section properties table.

Parameters

  • section_type (str) – The name of the section type to retrieve

  • section_size ("Angle" | "Channel" | "Circular" | "Rectangular" | "WF") – The name of the section size to retrieve

Returns

The physical properties for the section

Return type

dict

>>> size_options = ["WF 8 x 8.32", "WF 8 x 5.90", "WF 8 x 10.7"]
>>> size_selector = Input("size_a", "WF 8 x 10.7", input_type="select", select_options=size_options)
>>> section_properties = get_aluminum_section_properties("WF", size_selector.get_value())
>>> print(section_properties.get("A"))
9.12
>>> print(section_properties.get("Ix"))
110
get_steel_section_properties(section_type: str, section_size: str) → dict

Returns a dictionary with physical properties of the requested steel section type and size. This method expects the section_size to match one of the AISC_name sizes in the respective section properties table.

Parameters

  • section_type (str) – The name of the section type to retrieve

  • section_size ("Angle" | "DoubleAngle" | "Channel" | "Circular" | "Rectangular" | "T" | "WF") – The name of the section size to retrieve (AISC_name size)

Returns

The AISC defined physical properties for the section

Return type

dict

>>> size_options = ["W14X99", "W14X109", "W14X120"]
>>> size_selector = Input("size_a", "W14X99", input_type="select", select_options=size_options)
>>> section_properties = get_steel_section_properties("WF", size_selector.get_value())
>>> print(section_properties.get("A"))
29.1
>>> print(section_properties.get("Ix"))
1110
